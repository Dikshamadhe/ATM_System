# ATM System

A simple Python ATM System project created using basic Python concepts.

## Features

* Check Account Balance
* Deposit Money
* Withdraw Money
* Transaction History
* Change ATM PIN
* Exit System

## Technologies Used

* Python
* VS Code
* Git
* GitHub

## Python Concepts Used

* Variables
* if elif conditions
* while loop
* nested conditions
* list
* user input
* arithmetic operators

## How to Run

```python
python atmsystem.py
```

## Sample Menu

```text
1. Check Balance

6. Exit
```

## Project Description

This ATM System project allows users to:

* verify PIN
* deposit money
* withdraw money
* check balance
* view transaction history
* change ATM PIN

This project is useful for Python beginners to practice logic building and conditions.
